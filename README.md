# Course Enrollment System and Analytics System(Python OOP Project)

Project Overview: A simple and beginner-friendly Object-Oriented Programming (OOP) project in Python that demonstrates how classes, objects, constructors, inheritance, encapsulation, and method overriding work together to build a real-world application.
This project manages Students, Courses, and Enrollments, allowing users to add students, add courses, enroll students, and view enrollment details.


__________________________________________________________________________________________________________________________________________________________________________________________________________________
Features : 
* Add new students with basic information
* Add new courses
* Enroll a student into a course
* View all students, courses, or enrollments
* Demonstrates real-world OOP concepts
* Clean, modular, and reusable Python code


__________________________________________________________________________________________________________________________________________________________________________________________________________________
OOP Concepts Used :
* Classes & Objects : Separate classes for Student, Course, and Enrollment.
* Constructors (__init__) : Used to initialize class variables.
* Encapsulation : Used getter and setter methods to control access to private data.
* Inheritance : Created child classes to extend functionality when needed.
* Composition : Enrollment object combines both Student and Course objects.


 __________________________________________________________________________________________________________________________________________________________________________________________________________________
 Learning Outcomes :
* How to design real-life systems using OOP
* How to create clean and reusable class structures
* How to use inheritance, encapsulation in Python
* How object relationships work in real-world applications
